# fastify-prettier
> A simple and lightweight beautifier plugin for [Fastify](https://github.com/fastify/fastify).

[![NPM](https://nodei.co/npm/fastify-prettier.png)](https://nodei.co/npm/fastify-prettier/)

`fastify-prettier` has support of beautifying responses via query parameter and/or by using registered `prettier` decorator anywhere in the fastify server. It uses [prettier](https://github.com/prettier/prettier) under the hood and is capable of parsing/formatting anything that prettier can.

## Installation
`npm install fastify-prettier`
